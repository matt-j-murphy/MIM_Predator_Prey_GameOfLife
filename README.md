# **Game of Life: Predator vs Prey**
  
Predator vs Prey version of Conway's Game of Life
  
  
  
**Predators** = 1's (YELLOW)

**Prey** = 0.5's (TEAL)

**Dead/Non-Existent** = 0's (DARK BLUE)
  
  
  
**Instructions**:
- Run pred_check_neighbors AND pred_show_stuff first. 
- Then, run pred_game_of_life in order to begin the Predator_Prey game visualization
